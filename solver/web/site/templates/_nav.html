{# The fixed header (web-server-guide § The site): brand · nav · Actions · back · breadcrumbs · git · user menu, grouped by separators — one control surface, identical on every page and in both tiers. Links swap only #content (plus hx-push-url); logout is the only full-page exit. No hx-on / inline JS (CSP). `subject` is the switch. Signed out the header is *present and inert*: the visitor sees the shape of the place before being let into it. The brand and the user pill stay live — they are the two things that still do something (home, and the way in), so neither is dimmed. There is no theme control: the site is dark-only (site.css). #}
