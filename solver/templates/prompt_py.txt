Requirements:

1. The first line must be exactly: `#!/usr/bin/env python3.14`

2. Follow the template structure faithfully:
   - Shebang, `# -*- coding: utf-8 -*-`, module docstring (as in the template)
   - `from __future__ import annotations`
   - `solve(*, ...) -> <return_type>` with keyword-only typed parameters and a return type annotation
   - `main(**kwargs: Any) -> int` that parses `--runs=N` from argv, times `solve()`, and prints output
   - `if __name__ == "__main__": raise SystemExit(main(<kwarg>=<type>(argv[N]), ...))`

   existing approach.

4. Include `get_text_file` only when the solution must read a data file from the `resources`
   directory. Omit it otherwise.

5. The output line must be exactly: `print(f'{runs} {average} {result}')`

6. Use only the Python standard library. The only permitted third-party packages are `numpy`
   for numerical computation and `primesieve` if sieve-based prime generation is needed.
